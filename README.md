# Mini-Curso SASS

## O que é SASS?

Sass é uma linguagem de folhas de estilo que é interpretada ou compilada em Cascading Style Sheets (CSS). SassScript é a linguagem de script interpretada incorporada em Sass.

## Por que usar SASS?
